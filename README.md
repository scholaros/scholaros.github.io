## The Smart OS for Modern Classrooms

ScholarOS is a specialized Linux distribution built on the rock-solid foundation of [Edubuntu](https://edubuntu.org/), supercharged with local, offline Artificial Intelligence via [Ollama](https://ollama.com/). It is designed to empower students and teachers with cutting-edge technology while guaranteeing 100% data privacy.

### ✨ Features
* **🐧 Edubuntu Foundation:** Comes pre-loaded with the best educational software out of the box. From mathematics and science to art and programming, everything a student needs is ready to use without extra downloads.
* **🧠 Integrated AI (Ollama):** Features an embedded Ollama instance running locally. Students can interact with a powerful, specialized Large Language Model (LLM) designed strictly as an educational tutor.
* **🔒 100% Privacy & Offline Capable:** Since all AI models and software run entirely locally on the hardware, no student queries, data, or personal information ever leave the classroom or connect to external cloud servers.
* **🌍 Bilingual Design:** Built with support for both English and Portuguese environments.

### 📥 Download
You can download the latest ScholarOS `.iso` image directly from our official release folder:
**[Download ScholarOS ISO](https://drive.google.com/drive/folders/1p5p0nHEf7aJovvLhZJF8yXSMCjPJJbO3)**

### 💻 System Requirements
Because ScholarOS runs Large Language Models locally, it requires slightly more resources than a standard lightweight Linux distribution.
* **Minimum:** 64-bit Dual-Core processor, 8 GB RAM, 40 GB Storage (SSD recommended), Standard VGA.
* **Recommended:** Modern Quad-Core processor, 16 GB+ RAM, 60+ GB SSD/NVMe, Dedicated GPU (NVIDIA/AMD) for faster AI inference.

### 🚀 Installation
1.  **Download the ISO** from the link above.
2.  **Create a Bootable USB** using a tool like [BalenaEtcher](https://etcher.balena.io/) or [Rufus](https://rufus.ie/).
3.  **Boot your computer** from the USB drive.
4.  Follow the standard Edubuntu graphical installer to set up ScholarOS on your machine.

## O Sistema Operativo Inteligente para as Salas de Aula Modernas

O ScholarOS é uma distribuição Linux especializada, construída sobre a base sólida do [Edubuntu](https://edubuntu.org/), potenciada por Inteligência Artificial local e offline através do [Ollama](https://ollama.com/). Foi concebido para capacitar alunos e professores com tecnologia de ponta, garantindo em simultâneo 100% de privacidade dos dados.

### ✨ Características Principais
* **🐧 Base Edubuntu:** Vem pré-carregado com o melhor software educativo de raiz. Da matemática e ciências à arte e programação, tudo o que um aluno precisa está pronto a usar sem necessidade de transferências adicionais.
* **🧠 IA Integrada (Ollama):** Possui uma instância Ollama embutida a correr localmente. Os alunos podem interagir com um poderoso Modelo de Linguagem (LLM) especializado, concebido estritamente como um tutor educativo.
* **🔒 100% Privado e Offline:** Como todos os modelos de IA e software correm inteiramente localmente no hardware, nenhuma pergunta ou informação pessoal do aluno sai da sala de aula ou se liga a servidores na nuvem.
* **🌍 Design Bilingue:** Construído com suporte nativo para ambientes em Inglês e Português.

### 📥 Transferência
Pode descarregar a imagem `.iso` mais recente do ScholarOS diretamente da nossa pasta oficial:
**[Descarregar ISO do ScholarOS](https://drive.google.com/drive/folders/1p5p0nHEf7aJovvLhZJF8yXSMCjPJJbO3)**

### 💻 Requisitos de Sistema
Uma vez que o ScholarOS executa IA localmente, exige recursos ligeiramente superiores aos de um Linux padrão.
* **Mínimos:** Processador Dual-Core de 64 bits, 8 GB RAM, 40 GB Armazenamento (SSD recomendado), VGA padrão.
* **Recomendados:** Processador Quad-Core moderno, 16 GB+ RAM, 60+ GB SSD/NVMe, Placa gráfica dedicada (NVIDIA/AMD) para inferência de IA mais rápida.

### 🚀 Instalação
1.  **Descarregue o ISO** a partir do link acima.
2.  **Crie uma Pen USB de Arranque** usando uma ferramenta como o [BalenaEtcher](https://etcher.balena.io/) ou [Rufus](https://rufus.ie/).
3.  **Inicie o computador** a partir da pen USB.
4.  Siga o instalador gráfico padrão do Edubuntu para configurar o ScholarOS na sua máquina.
