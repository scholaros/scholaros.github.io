Início rápido

Verificar imagem
sha256sum -c ScholarOS-40-x86_64.iso.sha256
Criar USB (exemplo)
sudo dd if=ScholarOS-40-x86_64.iso of=/dev/sdX bs=4M status=progress && sync

Download

Descarregar ISO Live (x86_64) Checksums e Assinaturas

Notas de instalação

  1. Criar um USB de arranque (dd, ScholarOS Media Writer, Ventoy).
  2. Arrancar pelo USB e escolher “Iniciar ScholarOS Live” para testar sem instalar.
  3. Executar o instalador e escolher o disco, fuso horário e utilizador.
  4. Após o primeiro arranque, ativar repositórios de terceiros e atualizar o sistema.

Ativar ScholarOS Fusion (exemplo)

sudo dnf install https://download.scholaros.org/fusion/free/scholaros-fusion-free-release.noarch.rpm
sudo dnf install https://download.scholaros.org/fusion/nonfree/scholaros-fusion-nonfree-release.noarch.rpm
sudo dnf update -y

Substitua os valores de versão pelo número da sua versão do ScholarOS (ex.: 40).
